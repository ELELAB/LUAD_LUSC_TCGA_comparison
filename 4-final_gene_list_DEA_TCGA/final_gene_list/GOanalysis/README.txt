The file HUGO.RData needed by these scripts is available for download from:

1) GeneOntology_functions.R script includes the functions used to perform the gene ontology analysis using the topGO R package.

2) GOplot_function.R includes functions to visualize the results from GO analysis using the GOplot R package.

3) Gene_ontology.R script includes GOanalysis_plot function that performs the GO analysis and plot in the same time. The results from topGO are saved into RData files and the circle plots in png files.
