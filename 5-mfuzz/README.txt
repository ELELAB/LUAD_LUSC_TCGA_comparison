In this folder, the scripts to carry out mfuzz clustering are reported
Please first unzip the files LUAD/data/LUAD_all_consensus.csv.gz and LUSC/data/LUSC_all_consensus.csv.gz

To carry out the analysis open and run the scripts
(1)LUAD/scripts/build_Mfuzz_all_LUAD.R
(2)LUSC/scripts/build_Mfuzz_all_LUSC.R

NB the plots and csv file will be saved in LUAD/results and LUSC/results, respectively.
